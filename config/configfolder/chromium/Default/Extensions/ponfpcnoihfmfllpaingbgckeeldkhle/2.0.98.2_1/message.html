Upcoming change regarding permissions!

What will change

Currently the extension has permissions for YouTube™ (*://www.youtube.com/*) and for my website (*://*.mrfdev.com/*). In the next release I'm gonna remove the permission for my website, and because of this your browser will disable the extension to let you know that permissions have changed. All you will have to do to continue to use the extension is to enable it, no need to reinstall it.

Why this change?

The permission for my website is only needed to insert a message in the How to use Enhancer for YouTube™ page when it is open upon install. I'm gonna do things differently so there is no need to require this permission anymore. It also proves to paranoiacs that my extension is not tracking users, and that unlike what some developers claim it is possible to interact with YouTube™ (including embedded videos) with the bare minimum permissions.
